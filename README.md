# Charisios Charisiadis - Portfolio Website

A modern, animated portfolio website built with HTML, CSS, and JavaScript showcasing projects and skills.

## Features

- **Responsive Design**: Adapts seamlessly to all screen sizes
- **Animated UI**: Smooth scrolling, glowing effects, and hover animations
- **Dark Theme**: Eye-catching cyan and purple gradient color scheme
- **Starfield Background**: Animated background with particle effects
- **Project Showcase**: Display of web projects with demo links
- **Contact Form**: Web3Forms integration for easy communication
- **Skills Display**: Interactive skill tags with hover effects

## Sections

- **About**: Brief introduction
- **Projects**: Showcase of three projects:
  - A Calculator
  - Snake Game
  - Qequation (Quadratic Equation Guide)
- **Skills**: HTML, CSS, JavaScript, Python, Git, Problem Solving
- **Contact**: Contact form with email integration
- **Footer**: Social links (GitHub, Email, LinkedIn)

## Technologies Used

- HTML5
- CSS3 (Gradients, Animations, Flexbox, Grid)
- JavaScript
- Web3Forms API
